AVCP.NET
========

XML Generator AVCP / Adempimenti Legge 190 per i contratti della pubblica amministrazione


Strumento che consente di generare, in ambiente Microsoft.NET, i tracciati XML dei contratti delle pubbliche amministrazioni italiane, secondo la specifica AVCP:

http://www.avcp.it/portal/rest/jcr/repository/collaboration/Digital%20Assets/pdf/AllCom27.05.13SpecificeTecnichev1.0.pdf


Per testare l'applicazione è sufficiente lanciarla come un sito web con Visual Studio, la pagina iniziale genera e restituisce due file di esempio.
